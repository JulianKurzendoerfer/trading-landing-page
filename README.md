# Clean Landing Starter (Statisch, Render-ready)

Eine minimalistische, professionelle Landingpage mit Tailwind (CDN), DSGVO-Basics und sofortigem Deploy auf Render.

## Schnellstart
1. **Texte/Branding anpassen:** `index.html` öffnen und Logo/Abschnitte ändern.
2. **Rechtliches ergänzen:** `legal/impressum.html` & `legal/datenschutz.html` mit deinen echten Daten füllen.
3. **GitHub-Repo erstellen** und Code pushen.
4. **Render:** New → Static Site → GitHub Repo wählen → Build Command leer lassen → Publish Directory: `.` → Create.

> Optional: Nutze `render.yaml` und wähle bei Render „Blueprint“. 

## Design anpassen
- Primärfarbe via Tailwind-Klassen `bg-sky-600`/`text-sky-600` global ersetzen (z. B. `indigo`, `emerald`).
- Abschnitte duplizieren oder entfernen (Features, Preise, FAQ, Testimonials).
- Bilder im Ordner `assets/` ablegen und im HTML verlinken.

## Später erweitern
- Zu Next.js/React migrieren (z. B. Komponenten, Blog, Formulare).
- Kontaktform mit Formspree/Netlify Forms (no‑backend) oder eigenem Backend.
